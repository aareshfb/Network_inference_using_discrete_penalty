### Files

#### `simulation_scaleFree.R` : main script to navigate the MST as before, and generate the category-specific networks at each node of the population graph by perturbing the global network. Saves the global and local precision matrices for each population, and simulates Normally distributed counts for each categorical population

#### `perturb-strength.R` : includes an additional loop over varying perturbation strengths between the two local networks for each population

#### `simulation_helpers.R` : helper functions
