# Power-Law Experiment Results

This directory contains the **output results** for power-law experiments across different methods.

## Description

Each subdirectory contains the output files generated by the corresponding method:

- **`code_L0/Output/`**: Results for **ELEM-0** (proposed method)  
- **`code_L1_matlab/`**: Results for **ELEM-1** 
- **Other folders** (`FASJEM/`, `GGL/`, `GRNBoost2/`, `JGL/`, `SILGGM/`, `jointGHS/`): Results for competing methods  

These outputs are used to compute evaluation metrics such as F1 score, and correspond to the results reported in the paper.
